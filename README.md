Overview
This repository contains the 8051 Assembly Language Programs (ALP) for the Continuous Assessment (CA-1). All programs were simulated using the EdSim51DI simulator.

Case Studies Included
PRN Generation: Logically constructing the last four digits of my PRN (1294) using arithmetic instructions.
Restricted Comparison: Comparing two numbers without using CJNE, DJNZ, or SUBB.
Addressing Modes: Demonstrating the difference between Direct and Indirect addressing.
Logical Operations: Constructing the mobile number digits 0690 using bitwise logic.
Memory Compaction: In-place removal of FFH gaps in RAM.
Tools Used
EdSim51DI Simulator
